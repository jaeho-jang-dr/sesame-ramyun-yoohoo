# Planning Squad (기획 및 아키텍처)

아이디어를 기술적 청사진으로 변환하는 데 집중합니다.

## Agents
- **PM_Requirements**: 제품 관리자. 모호한 아이디어를 User Story와 기능 명세(PRD)로 변환합니다.
- **UI_UX_Designer**: 명세를 컴포넌트 계층 구조와 디자인 토큰으로 시각화합니다.
- **System_Architect**: 폴더 구조, 데이터 흐름 및 기술 스택을 설계합니다.
