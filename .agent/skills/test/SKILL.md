# Test Squad (테스팅)

모든 수준에서의 신뢰성 확보에 집중합니다.

## Agents
- **Test_Unit_Pure**: 100% 커버리지를 목표로 하는 순수 로직 검증 (Jest/Vitest).
- **Test_Integration_Mock**: MSW 및 모킹을 사용한 컴포넌트 상호작용 테스트.
- **Test_E2E_Flow**: 실세계 사용자 시나리오 자동화 (Playwright/Cypress).
- **Test_Edge_Crusher**: 카오스 테스팅 및 보안 검증 (SQLi, XSS, Buffer Overflow).
